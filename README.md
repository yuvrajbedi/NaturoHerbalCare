# Hairry Blossom by Naturo Herbal Care

A lightweight, responsive eCommerce website for a natural Ayurvedic hair care brand.

## Pages

- `index.html` - Home page with hero, featured products, benefits, why choose us, testimonials, FAQ, comparison, and tips.
- `shampoo.html` - Product page for Hairry Blossom Ayurvedic Shampoo.
- `hair-oil.html` - Product page for Hairry Blossom Herbal Hair Oil.
- `about.html` - Brand story and mission.
- `contact.html` - Contact form, WhatsApp link, and social links.

## Features

- Static HTML/CSS/JS for fast loading.
- Local product photos pulled from the Amazon listings, botanical visuals generated by `tools/generate-assets.mjs`, and a web-ready logo extracted from the supplied CDR using `tools/extract-cdr-logo.mjs`.
- Mobile responsive navigation.
- Shared cart with quantity controls and checkout form using `localStorage`.
- SEO titles, descriptions, keywords, Open Graph tags, and product structured data.

Open `index.html` in a browser to view the site.
